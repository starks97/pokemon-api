# execute in dev mode

1.- Clone repository
2.- Excute

```
yarn install
```
3.- Has Nest CLI installed globally

```
npm i -g @nestjs/cli
```
4.- Up DB

```
docker-compose up -d
```

5.- Rebuild DB
```
http://localhost:3000/api/v2/seed
```
